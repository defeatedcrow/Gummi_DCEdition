#### Gummi DCEdition ver1.0a for Minecraft 1.7.10 ####


# 前提Ver&Mod

・Minecraft1.7.10
・MincraftForge 1.7.10 10.13.2.1230 以上
　※Forge1230以上でも、ForgeのVerによっては不具合を生じることがあります。


# 使用方法

・MincraftForgeの導入後に作成されるmodsフォルダの中に、DLしたファイルを.jarファイルのまま置いてください。


# 配布DL先
 投稿フォーラム：http://forum.minecraftuser.jp/viewtopic.php?f=13&t=17657


# 概要

　このMODは、Andante氏の作成したGummiを勝手ながらdefeatedcrowが1.7.10向けに移植したものです。
　（フォーラム：http://forum.minecraftuser.jp/viewtopic.php?f=13&t=11895）

　"移植版"と銘打っていますが、ソースはdefeatedcrowにより新規に作りなおされています。
　したがって本家Gummi様とは構造も異なりますので、問い合わせの際は必ずdefeatedcrow宛にお願い致します。

　== 主な機能 ==

　●バニラの樹木から得られる果物

　　樫の木からリンゴが落ちるように、その他の樹木の葉からもさまざまな果物が落ちてきます。
　　ごくまれに金の果物が落ちてくることもあります。

　　白樺　　　：モモ
　　松　　　　：ブドウ
　　ジャングル：パインアップル
　　アカシア　：オレンジ

　　※オレンジは1.7移植に伴い新規に追加しました。


　●金の果物

　　金の果物は落ちてくる以外にも、クラフトによって（金リンゴのように）作成することができます。
　　
　　金塊＜金インゴット＜金ブロック　の順に効果が強力になり、名前も長くなります。


　●グミ

　　追加された果物を使ってグミを作ることができます。
　　グミはお腹が空いていない時も食べられ、主に体力を回復できるアイテムです。

　　・アップルグミ　リンゴ＋砂糖＋スライムボール
　　　ハート3個分の回復効果です。

　　・ピーチグミ　モモ＋砂糖＋スライムボール
　　　ハート2個分の回復効果と、水中で食べると酸素ゲージが回復します。

　　・グレープグミ　ブドウ＋砂糖＋スライムボール
　　　ハート2個分の回復効果と、負のポーション効果を解除します。

　　・パイングミ　パイン＋砂糖＋スライムボール
　　　30秒間の持続回復効果です。

　　・オレンジグミ　オレンジ＋砂糖＋スライムボール
　　　ハート2個分の回復効果と、空腹度を少し回復します。

　　・マグログミ　魚＋砂糖＋スライムボール
　　　10秒間の持続回復効果です。

　　・ミックスグミ　果物5種＋砂糖＋スライムボール
　　　ハート5個分の回復効果と、負のポーション効果の解除です。

　　・ミラクルグミ　金の果物5種＋砂糖＋スライムボール
　　　ハート10個分の回復効果と負のポーション解除の効果、
　　　さらに様々なポーション効果をプレイヤーに与えます。
